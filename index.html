    שרה חיה מזרחי
    נולדתי ב- 23.07.1943 בעיר פתח תקווה לשני הורי לאה ומיכאל מזרחי.
    בת יחידה אחרי שלושה בנים, להוריי הייתה בת נוספת שנפטרה ממחלה בגיל 5,
    כשנולדתי הוריי החליטו לקרוא לי על שמה שרה בתוספת השם חיה.
    את בעלי מזה 62 שנים, שמואל מאיה, הכרתי דרך טובה אחותו, שהייתה חברתי הטובה.
    בגיל 17 התחתנו כשהוא עדיין חייל בשרות סדיר. נולדו לנו 5 ילדים מהממים שזיכו אותנו
    ב-19 נכדים ו- 16 נינים שהם כולם מקור גאוותי.
    מעולם לא עבדתי מחוץ לבית. את כל זמני הקדשתי למשפחתי האהובה, התחביב העיקרי שלי.
    אהבתי מאוד לפנק אותם ולבשל להם, כך שכל אחד יקבל את מה שהוא הכי אוהב לאכול.
    כיום אנו מתגוררים במושב רינתיה, אליו הגענו לפני כ- 51 שנים. את ביתנו בנינו לבד, דבר שלקח לנו הרבה מאוד זמן, אך המאמץ השתלם, זכינו לחיות במקום שקט עם ציוץ ציפורים וקהילה תומכת.
    אני מאוד אוהבת את מפגשי החברים בהם אנו משחקים קלפים, צוחקים, מבלים ומעלים זיכרונות.
    אנו מקפידים לפחות שלוש פעמים בשנה לצאת יחדיו לנופש בארץ.

    Main differents:

    Web 1.0

    A version that allows reading and receiving information, including static web pages that use basic html, so that the site is built in the form of templates (Frameset) and the webmaster is the only one responsible for managing and updating it.

    Web 2.0

    An advanced version that allows writing and creating in addition to reading. including dynamic web pages, social networks, blogs and more. Uses XML for getting information from the server without having to load the page and RSS to receive updates from the site, thus enabling global management in social interaction.

    Web 3.0

    Allows reading, writing and executing, this version include also static HTML pages but with a dynamic display, Which includes turning the Internet into a database, focuses on making data available along with the semantics of the content and the links between them. for this purpose OWL uses unambiguous language to create conceptual frameworks. Which allows to create search engines, recommendation systems, etc.

    For query - "Sneakers"

    Web 1.0

    The search results will be static sites, which contain the search words of the query, the site content will be displayed for reading only, for example footwear sites that contain images of shoes and details.

    Web 2.0

    The search results will be dynamic sites, so that the site content allows reading and writing, for example the user can create a shopping cart and add products he is interested in, in addition he can write reviews on various products.

    Web 3.0

    The search results will be more varied and accurate, so that the sites that come back will have a semantic connection to the query words, for example returned Nike shoes that do not contain the word sneakers.
